# Data Directory

This folder contains the dataset used by the Cell Cycle Intelligence System.
The actual data files are **not tracked by Git** due to their size.

## How to Get the Data

### Option 1: Automatic (run the pipeline)

The pipeline (`main.py`) will attempt to prepare the dataset automatically
from the raw BBBC048 archive if it's present.

### Option 2: Manual Download

1. **Download BBBC048v1.zip** from the Broad Bioimage Benchmark Collection:

   https://data.broadinstitute.org/bbbc/BBBC048/BBBC048v1.zip

2. **Place the zip** in this `data/` directory:
   ```
   data/BBBC048v1.zip
   ```

3. **Extract** the contents into `data/BBBC048_raw/`:
   ```
   data/BBBC048_raw/
   ├── 1540408813.tar.gz
   ├── CCS_final_evaluation_scripts.tar.gz
   ├── CellCycle.mp4
   ├── CellCycle.zip
   ├── final.tar.gz
   ├── predictions.tar.gz
   └── scRNA.tar.gz
   ```

4. **Run the pipeline** — it will process the raw data and populate `data/synthetic/`:
   ```bash
   python main.py
   ```

## Directory Structure After Setup

```
data/
├── README.md              ← You are here
├── Ground_truth.lst       ← Ground truth labels (tracked)
├── BBBC048v1.zip          ← Raw download (ignored)
├── BBBC048_raw/           ← Extracted archive (ignored)
└── synthetic/             ← Processed images per phase (ignored, generated by pipeline)
    ├── G1/
    ├── S/
    ├── G2/
    ├── Prophase/
    ├── Metaphase/
    ├── Anaphase/
    └── Telophase/
```
